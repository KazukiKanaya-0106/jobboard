### AWS 構成

* **Web（SPA）**：S3 + CloudFront + ACM（HTTPS）
* **API（Hub）**：ECS Fargate（1サービス/1〜2タスク）+ ALB + ACM（HTTPS）
* **DB**：Aurora Serverless v2（PostgreSQL互換）最小ACU
* **設定/シークレット**：SSM Parameter Store（必要に応じて Secrets Manager 併用）
* **CI/CD**：GitHub Actions → OIDCでIAMロールassume → ECR push → ECS Service更新
* **監視**：CloudWatch（ログ/メトリクス）+ ALBアクセスログ（S3）
* **構成管理**：IaC（Terraform）で再現性担保
